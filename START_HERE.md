# 🚀 START HERE - Test Your App Locally

## ⚡ Super Quick Start (30 seconds)

### Option 1: Double-Click Method (Windows)
```
1. Double-click: start-local-demo.bat
2. Wait for 2 windows to open
3. In "Mobile App" window, press 'w'
4. Browser opens automatically
5. Done! ✅
```

### Option 2: Manual Method (Any OS)

**Terminal 1:**
```powershell
cd local-server
node server.js
```

**Terminal 2 (new window):**
```powershell
cd mobile-app
npm start
```

**Then press 'w' in Terminal 2**

## 🆘 Problems?

### Server won't start?
```powershell
cd local-server
npm install
node server.js
```

### App won't start?
```powershell
cd mobile-app
npm install
npm start
```

### Browser blank?
- Wait 30 seconds
- Refresh (Ctrl+R)
- Check both terminals are running
